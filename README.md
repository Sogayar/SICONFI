# SICONFI
Ferramenta em Python para coleta automatizada de dados fiscais municipais diretamente do Data Lake do Tesouro Nacional (SICONFI).  O projeto permite baixar dados em larga escala, utilizando  execução paralela, e salvar os resultados em arquivos CSV.  A aplicação possui uma interface gráfica simples que facilita o uso por usuários não técnicos.
